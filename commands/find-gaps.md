argument-hint: Optional scope or module to focus the gap analysis on

Run the find-gaps skill to identify design-to-code gaps, missing features, and compliance issues between the PRD, HTML prototypes, and actual implementation.

Focus areas:
- Screens and routes that are missing or incomplete
- API endpoints that don't match the spec
- Design system violations (colors, typography, spacing)
- Missing icons, images, or assets
- Auth/role checks that are missing
- Form validation gaps
- Mobile responsiveness issues

Report findings to `.project/status/` and update gap tracking docs.
